1. จงค้นคว้า Selector ที่ใชในขั้น ตอนที่8
td+td, th+th {
	Property: value
}
หมายถึงอะไร?

หมายถึง เงื่อนไขที่ทำงานก็ต่อเมื่อมีtagอยู่ติดกันตามที่กำหนดเช่น
td+td จะใช้งานได้ก็ต่อเมื่อมี
<td>A</td>
<td>B</td> <------ จะใช้งานได้

2. จากข้อ 1.จงหาตัวอย่างการใช้งาน Selector ชนิดนี้มาแสดงและอธิบายประกอบ

td+td, th+th {
	color: green;
}

<td>A</td>
